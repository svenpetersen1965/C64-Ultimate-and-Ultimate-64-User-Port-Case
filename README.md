# C64-Ultimate-and-Ultimate-64-User-Port-Case
On Jul. 10th, 2026, a wider version was uploaded, replacing the previous STL files and the case label.

The user port of the C64 Ultimate and the Ultimate 64 is on an external PCB, which ideally sits outside the actual computer to be accessible, connected with a ribbon cable. This ribbon cable may be routed through the cassette port cutout, as it is approximately aligned with the user port pin header.

As a result, the card edge connector and the pin header solder joints are exposed to the risk of short circuits and electrostatic discharge. The User Port case is designed to minimize these risks and to provide a neat, finished appearance.
